## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
- Nodemon

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `npm start` command.
- Register as a new user.
- Enjoy the app!
